# Système de supervision sonore LED

## Liste du matériel nécessaire

Cette liste correspond au matériel que nous avons acheté pour équiper la cantine de La Roche-Jaudy. Les liens des sites internet sur lesquels nous avons commandé le matériel est présent uniquement à titre indicatif afin de retrouver les références exacts. Il ne s'agit en aucun cas de publicité.

### Contrôleur principal

- **Raspberry Pi 4B (8GB)** - [lien]
  - Pi 4B car sortie audio Jack 3.5
  - 8GB de RAM pour une pérennité dans le temps
- **Dongle USB Zigbee 3.0 NABU CASA** - [lien]
  - Pour ajouter le support du protocole Matter Thread (capteurs)
  - Le support de Matter désactivera le Zigbee de ce Dongle
- **SONOFF - Clé USB Zigbee 3.0** - [lien]
  - Pour gérer le Zigbee (contrôleurs LED)

### LED

- **40m de bandeau LED RGB très lumineux** - [lien]
  - 4x 10m de ruban LED
  - 1200 Lumens/m
  - 60 LED/m
  - 14,4W/m (144 Watts par bandeau)
- **4 alimentations 150W** (car 144W par bandeau) - [lien]
- **4 contrôleurs Zigbee** (1 par alimentation) - [lien]

### Capteurs

- **4 modules Arduino Nano Matter** - [lien]
- **4 alimentations USB Type-C** - [lien]
- **4 détecteurs de bruit Grove 101020023** - [lien]
  - (Prévoir du matériel de soudure à étain)

### Audio

- **2 enceintes KOALA 6A BT** - [lien]
  - Puissante
  - Compacte
  - Bluetooth
  - Entrée microphone
  - Line Out pour connecter une autre enceinte
- **1 câble XLR 30m** - [lien]
  - Pour relier les enceintes
- **1 câble jack/jack 6.35** - [lien]
  - Pour relier le microphone à l'enceinte
- **1 câble jack/jack 3.5** - [lien]
  - Pour relier le Raspberry à l'enceinte
- **1 microphone** - [lien]
